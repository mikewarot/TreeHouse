TreeHouse - A structure build in a tree, for habitation

Everything is a node in a tree, with a sequentially assigned ID#

  Node
    ID -- assigned at creation
    Type - what kind of node is this?
    State - data internal to that type of node


Types of nodes

  Entry - this is a possible starting point for execution
    Why not enable executing ANY function, if it needs parameters, prompt the user for them

  Variable
    Name
    Type

  Constant
    Name
    Type

  Block - defines a set of statements

  If / Then / Else

  Case

  For Each

  For Variable

  While
